Blind Coding

Sum Even Or Odd

Given an integer n, your task is to compute the sum of all natural numbers from 1 to n (inclusive). If n is 0, the sum should be 0.

Example 1:

Input:

n = 1

Output: 1

For n = 1, the sum will be 1.

Example 2:

Input:

-n = 5

Output: 15

For n = 5, sum will be 15. 1 + 2 + 3 + 4 + 5 = 15.

Constraints:

1 <= n <= 10^4
