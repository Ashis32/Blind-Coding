Sum Of N Natural No.

Write a program to print the sum of 'N' Natural Numbers.

Example 1:

Input: 7

Output: 28

Explanation: The sum is 1+2+3+4+5+6+7=28

Example 2:

Input: 20

Output: 210

Explanation: The sum is 1+2+3+....+20=210

Constraints:

0 <= N <= 100000
